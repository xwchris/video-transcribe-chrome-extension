# Privacy Policy Draft

Videosays Chrome Extension submits public video links to Videosays so users can convert speech into transcript text.

The extension may collect and transmit:

- The video URL or share text the user explicitly submits.
- The user's Videosays API key for authenticated API requests.
- Transcription task metadata returned by Videosays, such as task ID, status, platform, title, and transcript result.

The extension stores the API key and last task metadata locally in Chrome storage. It does not read page contents in the background, sell user data, inject advertising, or use submitted data for personalized advertising.

Data is sent only to Videosays API endpoints at `https://api.videosays.com`.
